# Noise Monitor MVP

`Noise Monitor MVP` — Windows-приложение на Python для контроля громкости голоса через выбранный микрофон. Приложение показывает визуальные предупреждения поверх всех окон, чтобы можно было не шуметь во время игры, звонков или стрима.

Текущая версия: `0.1.0`

## Возможности

- работа в системном трее;
- защита от запуска второго экземпляра;
- выбор микрофона;
- сохранение настроек между перезапусками;
- относительная шкала громкости `0..100`;
- цветная экранная рамка:
  - желтая зона с плавным ростом толщины;
  - красная зона с максимальной толщиной;
  - плавное появление и затухание;
- вертикальная overlay-шкала громкости с числовым значением;
- выбор позиции overlay-шкалы;
- сворачиваемый график истории шума в окне настроек;
- история только в памяти приложения, с автоматической очисткой старых данных.

## Технологии

- `Python 3.11+`
- `PySide6`
- `PySide6.QtCharts`
- `sounddevice`
- `numpy`

## Установка

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install -e .
```

## Запуск

```powershell
noise-monitor
```

Или напрямую:

```powershell
python -m app.main
```

## Сборка exe

```powershell
python -m pip install pyinstaller
pyinstaller --noconfirm --clean --onefile --windowed --name NoiseMonitor --icon app\assets\noise_monitor.ico --hidden-import PySide6.QtCharts --collect-binaries sounddevice app\main.py
```

Готовый файл появится в каталоге `dist/`.

## Использование

1. Запусти приложение.
2. Выбери микрофон.
3. Настрой пороги `warning` и `critical`.
4. При желании включи overlay-шкалу и выбери ее позицию.
5. При желании раскрой график истории и выбери период `1`, `2` или `5` минут.
6. Сохрани настройки и подстрой чувствительность под свой голос.

## Как считается уровень

Приложение использует не абсолютные децибелы, а нормализованную относительную шкалу `0..100`, рассчитанную из RMS аудиосигнала с микрофона. Такой подход проще и стабильнее для разных микрофонов в рамках desktop MVP.

## Ограничения

- пока поддерживается только Windows;
- используется только основной экран;
- нет автозапуска вместе с Windows;
- нет многомониторного режима;
- нет автокалибровки под конкретный микрофон;
- история хранится только пока приложение запущено.
